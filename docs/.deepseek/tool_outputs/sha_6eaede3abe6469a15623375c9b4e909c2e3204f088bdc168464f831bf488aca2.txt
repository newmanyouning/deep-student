 14254│ 你可以按照以下顺序检查和修复：
 14255│ 
 14256│ ### 🔍 第一步：检查语言环境
 14257│ 打开终端，输入以下命令查看当前语言环境：
 14258│ ```bash
 14259│ locale
 14260│ ```
 14261│ 请关注 `LANG` 和 `LC_CTYPE` 的值，**它们必须包含 `zh_CN.UTF-8`**。如果结果显示为 `en_US.UTF-8` 或其他，就需要进行配置[citation:7]。
 14262│ 
 14263│ 修复方法：
 14264│ ```bash
 14265│ # 1. 重新配置语言环境，会弹出一个蓝色选择菜单
 14266│ sudo dpkg-reconfigure locales
 14267│ 
 14268│ # 2. 在菜单中，用空格键选中 `zh_CN.UTF-8 UTF-8`（前面会出现*号），按Tab键跳到“OK”并回车。
 14269│ # 3. 在接下来的菜单中选择 `zh_CN.UTF-8` 作为系统默认语言环境，回车确认[citation:2]。
 14270│ ```
 14271│ 
 14272│ ### ⚙️ 第二步：安装输入法框架和引擎
 14273│ Ubuntu系统常用的输入法框架是 `fcitx`（小企鹅输入法），它轻量且对中文支持好[citation:2]。
 14274│ 
 14275│ 安装命令：
 14276│ ```bash
 14277│ # 更新软件包列表并安装输入法框架、配置工具和谷歌拼音引擎[citation:2][citation:4][citation:6]
 14278│ sudo apt update
 14279│ sudo apt install fcitx fcitx-configtool fcitx-googlepinyin -y
 14280│ ```
 14281│ 
 14282│ ### 🌐 第三步：设置环境变量
 14283│ 为了让所有图形应用程序都知道使用 `fcitx`，需要在你的用户配置文件中设置环境变量。
 14284│ 
 14285│ **重要**：环境变量必须写入 `~/.profile` 文件（影响整个图形会话），而不是 `.bashrc`（只影响终端）[citation:2]。
 14286│ 
 14287│ ```bash
 14288│ # 将以下三行关键环境变量追加到 ~/.profile 文件末尾
 14289│ echo 'export GTK_IM_MODULE=fcitx' >> ~/.profile
 14290│ echo 'export QT_IM_MODULE=fcitx' >> ~/.profile
 14291│ echo 'export XMODIFIERS=@im=fcitx' >> ~/.profile
 14292│ ```
 14293│ 
 14294│ ### 🔄 第四步：重新启动
 14295│ 设置完成后，**必须重启系统或完全注销再登录**，以使所有配置生效。
 14296│ 
 14297│ ### 🚀 第五步：启动和配置输入法
 14298│ 登录后，你需要手动启动一次 `fcitx`：
 14299│ 1.  在应用程序菜单中搜索并打开 **Fcitx Configuration**（配置工具）。
 14300│ 2.  在配置界面左下角点击 **+** 号。
 14301│ 3.  取消勾选 **Only Show Current Language**（仅显示当前语言），然后在列表中找到并添加 **Google Pinyin**（谷歌拼音）[citation:6]。
 14302│ 4.  添加后，可以使用默认快捷键 **Ctrl + 空格** 来切换中英文输入[citation:2]。
 14303│ 
 14304│ ### 💡 第六步：特殊情况（Chrome/Edge浏览器）
 14305│ 如果你发现其他程序可以输入中文，但Chrome或Edge浏览器里无法输入，这可能是因为浏览器与新显示协议Wayland的兼容性问题[citation:5][citation:9]。
 14306│ 
 14307│ 临时解决方法是为浏览器创建一个特殊的启动命令（以Chromium为例）：
 14308│ ```bash
 14309│ # 在终端中，用以下命令启动浏览器可以强制使用兼容性更好的X11协议
 14310│ chromium --disable-features=UseOzonePlatform --ozone-platform=x11
 14311│ ```
 14312│ 如果这能解决问题，你可以修改桌面启动器，将上述参数加入浏览器的启动命令中，实现永久修复[citation:5]。
