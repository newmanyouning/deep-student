    14│ 在Ubuntu中，使用控制台安装下载的应用包（通常为.deb文件）可以使用多种命令。以下是一些常见的方法：
    15│ 
    16│ 1. 使用dpkg命令：
    17│    dpkg是Debian系列系统（包括Ubuntu）的包管理工具，可以直接安装.deb文件。
    18│    命令示例：sudo dpkg -i package_file.deb
    19│    注意：使用dpkg安装时可能会因为依赖问题而失败，此时可以运行sudo apt-get install -f来修复依赖。
    20│ 
    21│ 2. 使用apt命令：
    22│    apt是Ubuntu的高级包管理工具，也可以用来安装本地.deb文件。
    23│    命令示例：sudo apt install ./package_file.deb
    24│    注意：使用apt安装本地文件时，需要指定文件的路径（例如使用相对路径./package_file.deb）。
    25│ 
    26│ 3. 使用gdebi工具：
    27│    gdebi是一个可以处理.deb文件依赖关系的工具，如果没有安装，可以使用sudo apt install gdebi-core来安装。
    28│    命令示例：sudo gdebi package_file.deb
