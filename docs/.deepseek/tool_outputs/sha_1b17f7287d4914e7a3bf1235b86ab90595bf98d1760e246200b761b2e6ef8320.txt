   100│ ### 方案1：使用Maven/Gradle管理依赖（推荐）

   106│ <dependency>

   118│ ```

   120│ ### 方案2：创建包含依赖的Fat Jar/Uber Jar

   123│ 
   124│ **Maven Assembly插件配置**：
   125│ ```xml
   126│ <plugin>
   127│     <groupId>org.apache.maven.plugins</groupId>
   128│     <artifactId>maven-assembly-plugin</artifactId>
   129│     <version>3.3.0</version>
   130│     <configuration>
   131│         <descriptorRefs>
   132│             <descriptorRef>jar-with-dependencies</descriptorRef>
   133│         </descriptorRefs>
   134│         <archive>
   135│             <manifest>
   136│                 <mainClass>com.cburch.logisim.Main</mainClass>
   137│             </manifest>
   138│         </archive>
   139│     </configuration>
   140│     <executions>
   141│         <execution>
   142│             <id>make-assembly</id>
   143│             <phase>package</phase>
   144│             <goals>
   145│                 <goal>single</goal>
   146│             </goals>
   147│         </execution>
   148│     </executions>
   149│ </plugin>
   150│ ```
   151│ 
   152│ 然后运行：
   153│ ```bash
