   213│ #### 1. 词典匹配（Trie树+DAG）
   214│ ```python
   215│ # Trie节点定义
   216│ class TrieNode:
   217│     __slots__ = ['children', 'is_end']
   218│     def __init__(self):
   219│         self.children = {}
   220│         self.is_end = False
   221│ ```
   222│ - **最大匹配时间复杂度**：$O(n \cdot \max\_len)$  
   223│ - **缺陷**：无法处理未登录词，F1≤85%
   224│ 
   225│ #### 2. 隐马尔可夫模型（HMM）
   226│ **状态集**：$S = \{B, M, E, S\}$  
   227│ **观测序列**：$O = \{o_1, o_2, ..., o_n\}$  
   228│ **解码目标**：  
   229│ $$\arg\max_{s} P(s|o) = \prod_{i=1}^n P(o_i|s_i) \cdot \prod_{i=2}^n P(s_i|s_{i-1})$$  
   230│ **Viterbi复杂度**：$O(N^2T)$
   231│ 
   232│ #### 3. 深度学习方案（BiLSTM-CRF）
   233│ ```python
   234│ # CRF损失函数
   235│ loss = -log(exp(score_real) / sum(exp(score_all_paths))
   236│ ```
   237│ - **特征表示**：$h_t = [\overrightarrow{LSTM}(x_t); \overleftarrow{LSTM}(x_t)]$  
   238│ - **标签转移矩阵**：$A \in \mathbb{R}^{|S| \times |S|}$
