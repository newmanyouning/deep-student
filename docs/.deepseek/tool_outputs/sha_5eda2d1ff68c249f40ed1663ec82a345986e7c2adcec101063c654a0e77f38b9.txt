在 Ubuntu 22.04 服务器上，你可以使用多种命令来实时监控 GPU 和 CPU。这里有一份精选的工具列表，你可以根据你的需求和显卡类型来选择。

### 💻 CPU 监控常用命令
对于 CPU 监控，有几个经典且强大的命令行工具：

1.  **`htop`**：一个增强版的交互式进程查看器，比系统自带的 `top` 命令界面更友好，支持彩色显示和鼠标操作[citation:4]。
    ```bash
    # 安装
    sudo apt install htop -y
    # 运行
    htop
    ```

2.  **`dstat`**：一个多功能资源统计工具，可以帮你全面了解 CPU、内存、磁盘 I/O 和网络流量等系统资源[citation:8]。
    ```bash
    # 安装
    sudo apt install dstat -y
    # 运行 (默认显示CPU、磁盘、网络、分页和系统信息)
    dstat
    # 单独监控CPU和内存
    dstat --cpu --mem
    ```

### 🚀 GPU 监控常用命令
GPU 监控取决于你的显卡品牌，最推荐的是 **`nvtop`**，它堪称GPU版的 `htop`。

*   **对于 AMD 和 NVIDIA 显卡：** `nvtop` 是首选。它提供了一个直观的、类似 `htop` 的界面，可以实时显示 GPU 的整体利用率、显存使用情况、温度、功耗，甚至还能按进程查看占用[citation:7]。
    ```bash
    # 步骤1: 添加官方PPA源以获取最新版本
    sudo add-apt-repository ppa:flexiondotorg/nvtop
    # 步骤2: 更新软件包列表
    sudo apt update
    # 步骤3: 安装nvtop
    sudo apt install nvtop -y
    # 步骤4: 直接运行
    nvtop
    ```

*   **对于 NVIDIA 显卡：** 如果你安装了 NVIDIA 驱动，系统自带的 **`nvidia-smi`** 命令是最基础且可靠的选择。它不仅能查看 GPU 状态，还能进行一些管理操作。
    ```bash
    # 直接运行，默认每隔1秒刷新一次
    watch -n 1 nvidia-smi
    ```
    这里的 `watch` 命令让 `nvidia-smi` 每1秒刷新一次，从而实现实时监控的效果。

*   **对于 Intel 集成显卡：** 可以使用 `intel_gpu_top` 工具。
    ```bash
    # 安装
    sudo apt install intel-gpu-tools -y
    # 运行 (通常需要sudo权限)
    sudo intel_gpu_top
    ```

### ✨ 全能型选手：Glances
如果你希望在一个窗口中同时看到 CPU、内存、磁盘、网络以及 GPU 的所有信息，那么 **`glances`** 是一个非常棒的选择。它是一个用 Python 编写的跨平台监控工具，信息显示非常全面[citation:4]。
```bash
# 安装
sudo apt install glances -y
# 运行
glances
```
**请注意**：`glances` 的 GPU 监控功能通常需要调用 `nvtop` 或 `nvidia-smi` 等工具来实现，但在大多数情况下，它能很好地自动检测并显示 GPU 信息。

### 总结与建议
为了方便你选择，这里总结一下：

| 监控需求 | 推荐工具 | 安装命令 | 主要特点 |
| :--- | :--- | :--- | :--- |
| **CPU + GPU 一站式** | **`glances`** | `sudo apt install glances` | 信息最全面，一目了然[citation:4] |
| **GPU 专用 (AMD/NVIDIA)** | **`nvtop`** | `sudo add-apt-repository ppa:flexiondotorg/nvtop && sudo apt install nvtop` | 交互界面友好，支持按进程查看，功能强大[citation:7] |
| **CPU 专用** | **`htop`** | `sudo apt install htop` | 经典的增强型进程管理器[citation:4] |
| **NVIDIA 快速查看** | **`watch -n 1 nvidia-smi`** | (已随驱动安装) | 简单直接，无需安装新工具 |

你可以根据自己的习惯选择一两个工具配合使用。例如，用 `nvtop` 专注看 GPU，用 `htop` 专注看 CPU，或者直接用 `glances` 一览无余。
