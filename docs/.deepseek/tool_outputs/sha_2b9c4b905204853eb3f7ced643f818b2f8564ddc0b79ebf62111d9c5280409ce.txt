  2736│ ### **二次型展开核心步骤（纯文本逐步计算）**
  2737│ 设变量偏移量：  
  2738│ - dx = x - μₓ  
  2739│ - dy = y - μᵧ  
  2740│ 
  2741│ **步骤1：计算中间向量**  
  2742│ 逆矩阵乘以偏移向量：  
  2743│ ```
  2744│ [ 1/σₓ²     -ρ/(σₓσᵧ) ]   [ dx ]   [ (dx/σₓ²) - (ρ·dy)/(σₓσᵧ) ]
  2745│ [                     ] × [    ] = [                         ]
  2746│ [ -ρ/(σₓσᵧ)  1/σᵧ²    ]   [ dy ]   [ (dy/σᵧ²) - (ρ·dx)/(σₓσᵧ) ]
  2747│ ```
  2748│ 
  2749│ **步骤2：计算点积**  
  2750│ 偏移向量点乘中间向量：  
  2751│ ```
  2752│ [dx, dy] • [上述结果] = dx×[(dx/σₓ²) - (ρ·dy)/(σₓσᵧ)] + dy×[(dy/σᵧ²) - (ρ·dx)/(σₓσᵧ)]
  2753│ ```
  2754│ 
  2755│ **步骤3：展开乘积**  
  2756│ = dx×(dx/σₓ²) - dx×(ρ·dy)/(σₓσᵧ) + dy×(dy/σᵧ²) - dy×(ρ·dx)/(σₓσᵧ)  
  2757│ = (dx²/σₓ²) - (ρ·dx·dy)/(σₓσᵧ) + (dy²/σᵧ²) - (ρ·dx·dy)/(σₓσᵧ)  
  2758│ 
  2759│ **步骤4：合并同类项**  
  2760│ = (dx²/σₓ²) + (dy²/σᵧ²) - 2ρ(dx·dy)/(σₓσᵧ)  
  2761│ 
  2762│ **步骤5：添加系数**  
  2763│ 最终二次型 = 1/(1-ρ²) × [ (dx²/σₓ²) - 2ρ(dx·dy)/(σₓσᵧ) + (dy²/σᵧ²) ]

  2776│ ```
  2777│ 二元正态分布密度函数 = 
  2778│ ┌───────────────────────────────┐
  2779│ │         归一化系数           │  ×  ┌───────────────────────────────────────┐
  2780│ └───────────────────────────────┘    │              指数函数                 │
  2781│                                     └───────────────────────────────────────┘
  2782│ 
  2783│ 其中：
  2784│ 1. 归一化系数 = 1 / [2π × σₓ × σᵧ × √(1-ρ²)]
  2785│    - σₓ：X的标准差
  2786│    - σᵧ：Y的标准差
  2787│    - ρ：相关系数
  2788│    - √(1-ρ²)：相关性校正因子
  2789│ 
  2790│ 2. 指数函数 = exp{ -1/[2(1-ρ²)] × [三项式] }
  2791│    三项式 = 
  2792│       ┌──────────────────┐   ┌───────────────────────────┐   ┌──────────────────┐
  2793│       │ (x-μₓ)²          │   │ (x-μₓ)(y-μᵧ)            │   │ (y-μᵧ)²          │
  2794│       │ ───────    -   2ρ │ × │ ─────────────    +       │ × │ ───────          │
  2795│       │   σₓ²             │   │    σₓσᵧ                 │   │   σᵧ²            │
  2796│       └──────────────────┘   └───────────────────────────┘   └──────────────────┘
  2797│ ```

  2799│ ### **物理意义说明**  
  2800│ 1. **归一化系数**  
  2801│    - 确保整个概率密度积分为1  
  2802│    - 当ρ=0时简化为：1/(2πσₓσᵧ)  
  2803│ 
  2804│ 2. **指数项的三部分**  
  2805│    - 第一部分：(x-μₓ)²/σₓ² → X方向偏离均值的程度  
  2806│    - 第三部分：(y-μᵧ)²/σᵧ² → Y方向偏离均值的程度  
  2807│    - 中间部分：-2ρ(x-μₓ)(y-μᵧ)/(σₓσᵧ) → X和Y相关性的调整项  
  2808│ 
  2809│ 3. **相关系数ρ的作用**  
  2810│    - ρ=0：交叉项消失 → 两个独立的正态分布  
  2811│    - ρ>0：当x和y同向偏离时，整体值减小 → 正相关  
  2812│    - ρ<0：当x和y反向偏离时，整体值减小 → 负相关  
  2813│    - |ρ|接近1：1/(1-ρ²)增大 → 概率分布沿对角线收缩

  2817│ ### **应用示例计算（ρ=0.5）**  
  2818│ 已知参数：  
  2819│ - μₓ=0, μᵧ=0  
  2820│ - σₓ=1, σᵧ=2  
  2821│ - ρ=0.5  
  2822│ 求点(1,2)的密度  
  2823│ 
  2824│ 1. **计算归一化系数**  
  2825│    1/(2π×1×2×√(1-0.5²)) = 1/(4π√0.75) ≈ 1/(4×3.1416×0.866) ≈ 0.0919  
  2826│ 
  2827│ 2. **计算三项式**  
  2828│    = (1-0)²/1² - 2×0.5×(1-0)(2-0)/(1×2) + (2-0)²/2²  
  2829│    = 1 - 1 + 1 = 1  
  2830│ 
  2831│ 3. **计算指数项**  
  2832│    exp{ -1/[2(1-0.25)] × 1 } = exp(-1/1.5) ≈ e⁻⁰·⁶⁶⁶⁷ ≈ 0.5134  
  2833│ 
  2834│ 4. **最终密度**  
  2835│    0.0919 × 0.5134 ≈ 0.0472
