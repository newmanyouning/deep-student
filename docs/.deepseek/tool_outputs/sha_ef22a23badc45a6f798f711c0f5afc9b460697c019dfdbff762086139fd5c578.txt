   735│ ### 4. 提取单元格间引用关系
   736│ 这是你最关心的功能。可以用 openpyxl 自带的 **Tokenizer** 来解析公式中的引用[reference:4]，也可以使用 **pycel** 这个更专业的库来构建完整的依赖图[reference:5]。
   737│ 
   738│ **方法一：用 openpyxl 原生解析**
   739│ ```python
   740│ from openpyxl.formula import Tokenizer
   741│ 
   742│ tokens = Tokenizer('=SUM(A1:A10)+B5')
   743│ for token in tokens.items:
   744│     if token.type == 'OPERAND' and ':' in token.value:  # 区域引用
   745│         print(f'引用了区域: {token.value}')
   746│     elif token.type == 'OPERAND' and token.subtype == 'RANGE':
   747│         print(f'引用了单元格: {token.value}')
   748│ ```
   749│ 
   750│ **方法二：用 pycel 构建完整依赖图**
   751│ ```python
   752│ from pycel.excelcompiler import ExcelCompiler
   753│ 
   754│ # 解析整个工作簿，构建依赖关系图
   755│ excel_model = ExcelCompiler('report.xlsx')
   756│ # 获取所有依赖关系
   757│ dependencies = excel_model.dependency_graph
   758│ # 获取某个单元格的引用来源（precedents）
   759│ precedents = excel_model.get_precedents('Sheet1!C1')
   760│ ```

   788│ ## 二、AI 辅助开发时，上下文窗口应该选多长？
   789│ 
   790│ 既然你提到在 Mac 上用 AI 辅助开发，那么核心问题是：**开发这个工具的过程中，AI 模型需要处理多长的上下文？**
   791│ 
   792│ 要回答这个问题，需要分析你的使用场景分成了哪些不同阶段：
   793│ 
   794│ ### 场景分类
   795│ 
   796│ | 使用场景 | 上下文需求 | 具体内容 |
   797│ |----------|-----------|----------|
   798│ | **日常咨询**（单文件/单函数生成） | 32K-128K | 问一个具体问题、生成一个函数、解释一段代码 |
   799│ | **整体项目开发** | 128K-256K | 理解整个工具的架构、生成跨模块代码、调试复杂问题 |
   800│ | **大文件处理** | 256K-1M | 上传整个工具的核心代码文件（超过 500 行）、分析项目结构 |
   801│ 
   802│ ### 实际需求分析
   803│ 
   804│ 以你这个工具为例，核心代码文件的规模预估：
   805│ 
   806│ - `excel_parser.py`（解析模块）：约 200-300 行
   807│ - `gui_app.py`（界面模块）：约 300-500 行（tkinter 界面代码比较长）
   808│ - `dependency_manager.py`（依赖管理）：约 150-250 行
   809│ - 配置文件和辅助模块：约 100 行
   810│ 
   811│ **一个核心文件大约 300-500 行，约 10,000-15,000 tokens**。这意味着：
   812│ 
   813│ 1. **日常开发**：128K 完全够用，甚至可以一次性把整个项目的所有核心文件都塞进上下文。
   814│ 2. **复杂调试**：如果需要 AI 同时分析多个文件之间的调用关系，256K 更从容。
   815│ 3. **大文件上传**：如果你需要上传一个完整的 Excel 解析结果（比如一个超大 Excel 文件的结构）让 AI 分析，可能需要 1M 窗口。
   816│ 
   817│ ### 实用建议
   818│ 
   819│ > **选择 128K-256K 的模型性价比最高，日常够用，偶尔复杂任务也不会受限。**
