   242│  步骤：
   243│  1. 卸载fcitx及相关包
   244│  2. 安装ibus和拼音引擎
   245│  3. 恢复环境变量设置（移除fcitx的环境变量，或者设置为ibus）
   246│  4. 配置ibus为默认输入法
   247│  5. 重启系统或重新登录
   248│ 
   249│  注意：在树莓派5的Ubuntu中，默认的输入法框架可能是ibus，但也不一定。我们按照典型Ubuntu的配置来操作。
   250│ 
   251│  具体操作：
   252│ # 还原到默认输入法（IBus）的完整步骤
   253│ 
   254│ ## 1. **完全移除Fcitx**
   255│ 
   256│ ```bash
   257│ # 1. 完全卸载Fcitx及相关组件
   258│ sudo apt remove --purge fcitx* -y
   259│ 
   260│ # 2. 清除依赖包（可选）
   261│ sudo apt autoremove -y
   262│ 
   263│ # 3. 清除配置文件和残留
   264│ rm -rf ~/.config/fcitx
   265│ rm -rf ~/.local/share/fcitx
   266│ ```
   267│ 
   268│ ## 2. **安装IBus和中文输入法**
   269│ 
   270│ ```bash
   271│ # 1. 安装IBus框架
   272│ sudo apt update
   273│ sudo apt install ibus ibus-gtk ibus-gtk3 ibus-qt4 -y
   274│ 
   275│ # 2. 安装中文输入法引擎
   276│ sudo apt install ibus-pinyin ibus-libpinyin ibus-sunpinyin -y
   277│ 
   278│ # 3. 安装完整的IBus配置工具
   279│ sudo apt install ibus-clutter ibus-el ibus-dbus -y
   280│ ```
   281│ 
   282│ ## 3. **清除Fcitx环境变量**
   283│ 
   284│ ```bash
   285│ # 1. 编辑~/.profile文件，删除之前添加的Fcitx环境变量
   286│ nano ~/.profile
   287│ # 找到并删除这三行：
   288│ # export GTK_IM_MODULE=fcitx
   289│ # export QT_IM_MODULE=fcitx
   290│ # export XMODIFIERS=@im=fcitx
   291│ 
   292│ # 2. 清除系统环境变量设置
   293│ sudo nano /etc/environment
   294│ # 删除以下行（如果存在）：
   295│ # GTK_IM_MODULE=fcitx
   296│ # QT_IM_MODULE=fcitx
   297│ # XMODIFIERS=@im=fcitx
   298│ 
   299│ # 3. 同样检查~/.bashrc文件
   300│ nano ~/.bashrc
   301│ # 删除相同的环境变量行
   302│ ```
   303│ 
   304│ ## 4. **设置IBus为默认输入法**
   305│ 
   306│ ```bash
   307│ # 1. 启动IBus设置
   308│ im-config
   309│ 
   310│ # 在弹出的对话框中选择"ibus"
   311│ # 或者使用命令设置：
   312│ sudo im-config -s ibus
   313│ 
   314│ # 2. 启动IBus守护进程
   315│ ibus-daemon -drx
   316│ 
   317│ # 3. 运行IBus配置工具
   318│ ibus-setup
   319│ ```

   321│ ## 5. **配置IBus输入法**
   322│ 
   323│ 在IBus设置界面中：
   324│ 1. 点击"Input Method"选项卡
   325│ 2. 点击"Add"按钮

   327│ 4. 点击"Add"

   330│ ## 6. **设置自动启动IBus**
   331│ 
   332│ ```bash
   333│ # 1. 创建IBus自动启动项
   334│ mkdir -p ~/.config/autostart
   335│ cat > ~/.config/autostart/ibus.desktop << EOF
   336│ [Desktop Entry]
   337│ Type=Application
   338│ Name=IBus Daemon
   339│ Exec=ibus-daemon -drx
   340│ Comment=IBus Input Method Framework

   342│ EOF
   343│ 
   344│ # 2. 或者使用GNOME设置
   345│ # 打开"应用程序" → "优化" → "开机启动程序" → 添加IBus
   346│ ```
