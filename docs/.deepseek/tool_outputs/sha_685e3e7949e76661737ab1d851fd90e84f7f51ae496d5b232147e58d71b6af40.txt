    23│ 现代计算机的 CPU（中央处理器）是一个高度集成的复杂芯片，通常由以下几个核心组成部分协同工作，共同执行程序指令和处理数据：
    24│ 
    25│ 1.  **控制单元**
    26│     *   **功能：** CPU 的“指挥中心”。
    27│     *   **作用：** 负责从内存中**提取指令**，**解码**指令以确定需要执行什么操作，然后**协调**其他所有部件（ALU、寄存器等）按照正确的顺序和时序执行这些操作。它生成控制信号来管理数据在 CPU 内部及 CPU 与内存、I/O 设备之间的流动。
    28│ 
    29│ 2.  **算术逻辑单元**
    30│     *   **功能：** CPU 的“计算引擎”。
    31│     *   **作用：** 执行所有的**算术运算**和**逻辑运算**。
    32│     *   **算术运算：** 加、减、乘、除等。
    33│     *   **逻辑运算：** 与、或、非、异或、移位、比较等。ALU 的操作结果通常会设置状态标志（如零标志、进位标志、溢出标志等），这些标志会被后续指令（如条件跳转）使用。
    34│ 
    35│ 3.  **寄存器**
    36│     *   **功能：** CPU 内部**超高速、容量极小**的存储单元。
    37│     *   **作用：** 用于临时存放当前正在被处理或即将被处理的**指令、数据、地址和状态信息**。访问速度远高于内存（RAM）。
    38│     *   **主要类型：**
    39│         *   **通用寄存器：** 存放操作数和中间计算结果，可被程序员或编译器灵活使用。
    40│         *   **指令指针：** 存放下一条要执行的指令在内存中的地址。
    41│         *   **程序状态字：** 存放 CPU 当前的运行状态信息（如各种状态标志位、中断允许位、工作模式位等）。
    42│         *   **栈指针：** 指向内存中当前栈顶的位置，用于函数调用、参数传递、局部变量存储等。
    43│         *   **段寄存器：** 在某些架构（如 x86）中用于管理内存分段。
    44│ 
    45│ 4.  **缓存**
    46│     *   **功能：** CPU 内部或紧邻 CPU 的**高速存储器**。
    47│     *   **作用：** 作为 CPU 与相对较慢的主内存之间的**缓冲区**，存储最近或频繁使用的指令和数据副本，**显著减少 CPU 等待内存数据的时间**。
    48│     *   **层级：** 通常有多级（L1、L2、L3）。
    49│         *   **L1 缓存：** 速度最快，容量最小（几 KB 到几十 KB），通常集成在 CPU 核心内部，分成指令缓存和数据缓存。
    50│         *   **L2 缓存：** 速度稍慢于 L1，容量较大（几百 KB 到几 MB），可能被单个核心独占或多个核心共享。
