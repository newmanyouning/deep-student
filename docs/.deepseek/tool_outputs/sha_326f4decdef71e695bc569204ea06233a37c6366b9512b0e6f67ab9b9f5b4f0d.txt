    28│ ### ✨ 核心特点
    29│ 它厉害的地方不是“模型更聪明”，而是**工程化**做得很好[citation:1]：
    30│ - **团队协作**：内部有不同角色分工，有的负责研究，有的负责执行[citation:1]。
    31│ - **自动规划**：执行复杂任务（如重构）时会先生成计划给你确认，避免乱改[citation:2][citation:3]。
    32│ - **上下文管理**：自动压缩对话历史，防止聊着聊着就“失忆”了，能省不少Token[citation:1]。
    33│ - **权限控制**：改文件前会问你要权限，安全感拉满[citation:1][citation:3]。
    34│ - **萌宠彩蛋**：内置“Buddy”系统，可以养一只电子宠物陪你写代码，还能提升用户粘性[citation:3]。

    36│ ### 🛠️ 使用步骤（Windows/Mac/Linux）
    37│ 使用前需要准备**Anthropic API Key**，或者购买Pro/Max订阅[citation:8]。如果没有，也可以配置三方代理[citation:2][citation:10]。
    38│ 
    39│ 1. **安装Node.js**
    40│    需要18.0或更高版本，终端输入`node -v`检查[citation:2][citation:4]。
    41│ 2. **全局安装**
    42│    ```bash
    43│    npm install -g @anthropic-ai/claude-code
    44│    ```
    45│    Mac用户也可以用Homebrew：`brew install --cask claude-code`[citation:4]。
    46│ 3. **配置密钥（最关键）**
    47│    **千万不要把Key写死在代码里！**
    48│    建议用配置文件（`~/.claude/settings.json`）：
    49│    ```json
    50│    {
    51│      "env": {
    52│        "ANTHROPIC_AUTH_TOKEN": "sk-your-key-here",
    53│        "ANTHROPIC_BASE_URL": "https://api.anthropic.com"
    54│      }
    55│    }
    56│    ```
    57│    *注意：如果使用三方代理，记得修改`ANTHROPIC_BASE_URL`*[citation:10]。
    58│ 4. **启动使用**
    59│    进入项目目录，输入`claude`回车即可[citation:4][citation:8]。
    60│ 
    61│ ### 🚀 三种使用方式
    62│ - **终端直接问**：`claude "修复这个bug"`，也可以加`-p`参数一次性执行[citation:4][citation:6]。
    63│ - **VS Code插件**：搜索“Claude Code”安装，装好后可以直接右键选中代码，让Claude帮你解释或修改，改完还能用**Diff视图**直观看到变化，比纯命令行好用很多[citation:2][citation:8][citation:9]。
    64│ - **高级集成**：支持MCP协议，可以给它联网搜索、连接数据库等新能力[citation:2][citation:6]。
    65│ 
    66│ ### 💡 给新手的建议
    67│ - **先开Plan模式**：执行命令前输入`/plan`，让它只出方案不改代码，确认了再动手，避免翻车[citation:2]。
    68│ - **用好`/init`**：在新项目跑一下这个命令，生成`CLAUDE.md`文件，让它记住项目规则[citation:6][citation:7]。
    69│ - **巧用快捷键**：在IDE里把生成代码设成快捷键（如`Ctrl+Alt+C`），效率直接起飞[citation:8]。
    70│ - **选对模型**：简单任务用**Haiku**，日常用**Sonnet**（性价比高），特别复杂再请**Opus**，别一上来就用最强的，费钱[citation:8]。
