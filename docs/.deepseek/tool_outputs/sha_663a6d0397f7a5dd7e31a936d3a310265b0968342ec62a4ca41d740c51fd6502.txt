  2916│ ### 14.3 最终专家建议
  2917│ 
  2918│ 1. **少即是多**：不要试图记住所有快捷键，先掌握 20 个最常用的，然后根据你的工作流逐步增加。
  2919│ 2. **自定义优先**：如果某个默认快捷键不顺手，使用 AutoHotkey 或 Karabiner 重新映射。你的键盘应该适应你，而不是相反。
  2920│ 3. **跨平台一致性**：如果你经常切换系统，可以考虑将 macOS 的修饰键映射为与 Windows 一致（如将 `⌘` 映射为 `Ctrl`），但需要适应。
  2921│ 4. **定期复习**：每季度花 30 分钟复习快捷键列表，删除不用的，添加新学的。
  2922│ 5. **教学相长**：向同事分享你学到的快捷键，教学是最有效的学习方式。
  2923│ 
  2924│ > [!TIP]
  2925│ > **最后的专家提示**：真正的高效不是“按得多快”，而是“按得准”。慢一点，准确一点，肌肉记忆会自然形成。祝你在键盘之路上越走越远！

  3037│ 
  3038│ - **Windows 11 vs Windows 10**：`Win + Z`（贴靠布局）为 Windows 11 新增；`Win + W` 在 Win10 中是 Windows Ink 工作区，Win11 中是小组件
