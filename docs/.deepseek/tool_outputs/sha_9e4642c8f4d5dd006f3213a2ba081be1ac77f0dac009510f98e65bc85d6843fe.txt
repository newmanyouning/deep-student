    21│ 在 Mac 上卸载应用程序通常有三种主要方法，根据你的需求选择即可：
    22│ 
    23│ ### 方法一：通过“访达”直接删除（最常用）
    24│ 适用于大多数从网站下载或通过拖拽安装的软件。
    25│ 
    26│ 1.  打开 **访达**，在侧边栏点击 **“应用程序”**。
    27│ 2.  找到你想删除的应用图标。
    28│ 3.  **直接拖拽**该图标到废纸篓；或者**右键点击**图标，选择 **“移到废纸篓”**。
    29│ 4.  输入你的电脑密码（如果需要），然后右键点击 Dock 栏的 **“废纸篓”**，选择 **“清倒废纸篓”**。
    30│ 
    31│ ### 方法二：使用“启动台”删除
    32│ 适用于从 **App Store** 下载的应用。
    33│ 
    34│ 1.  点击 Dock 栏上的 **“启动台”** 图标（火箭形状）。
    35│ 2.  找到你想删除的应用。
    36│ 3.  **长按**该应用图标（触控板用力按压或鼠标按住不放），直到图标开始抖动。
    37│ 4.  点击图标左上角的 **“X”** 按钮，点击“删除”确认即可。
    38│ 
    39│ ### 方法三：使用软件自带的卸载程序
    40│ 部分大型软件（如 Adobe 全家桶、Office、杀毒软件等）不推荐直接拖进废纸篓，否则容易残留大量文件。
    41│ 
    42│ 1.  打开 **访达** > **应用程序**。
    43│ 2.  查看该软件的文件夹里是否有名为 **“Uninstaller”** 或 **“卸载”** 的程序。
    44│ 3.  双击运行该程序，按照提示完成卸载。
