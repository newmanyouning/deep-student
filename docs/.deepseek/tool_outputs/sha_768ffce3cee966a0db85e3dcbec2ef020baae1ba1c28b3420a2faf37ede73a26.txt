MCP 是个多义词，根据你提问的语境，它最可能指 **Model Context Protocol**，尤其是在技术、AI、编程圈子里。

简单来说：

**MCP 是让 AI 能直接“插”上外部工具和数据源的通用协议。**

它是 Anthropic（Claude 的母公司）推出的一套开放标准。它的核心作用是解决一个问题：大语言模型很聪明，但它本身是个“大脑”，没有手和眼睛，出不了门，查不了你电脑里的文件，也访问不了实时数据。

MCP 就相当于给这个大脑装上了 **USB 接口**：

- **对于 AI（如 Claude）**：它就像一个万能插座，想用什么工具（查文档、看 GitHub、发邮件），按协议插上就行，不用为每个工具单独写代码。
- **对于开发者/用户**：你可以把自己本地的文件夹、数据库或 Google Drive 通过 MCP 服务器共享给 AI，AI 就能直接读取里面的内容帮你干活。

**举个例子帮你理解：**
没有 MCP 的时候，想让 AI 看一份 100 页的 PDF，你得手动把文件内容复制粘贴进对话框（还受字数限制）。
有了 MCP，你只需要说：“看看我桌面上那个项目文档里的预算表”，AI 会通过 MCP 直接读取那个 PDF，然后回答你。

MCP（Model Context Protocol）的核心原理是：**它是一个基于客户端-服务器模型的开放标准协议，采用JSON-RPC作为通信语言，通过定义“工具”、“资源”和“提示词”等标准原语，为大模型与外部数据源之间搭建了一座安全、标准化的桥梁**[reference:0][reference:1]。

### ⚙️ MCP的核心组件与工作机制

*   **MCP 主机 (Host)**：AI交互的协调中心，如Claude Desktop、Cursor等。它负责管理各个客户端、执行安全策略和用户授权，并聚合来自不同服务器的上下文信息[reference:2]。
*   **MCP 客户端 (Client)**：由主机创建，每个客户端与一个服务器建立独立的一对一连接。它负责协议协商、消息路由和状态维护，确保不同服务器的连接相互隔离[reference:3][reference:4]。
*   **MCP 服务器 (Server)**：具体能力的提供者。每个服务器专注于一项特定功能，如文件操作、数据库查询等。它可以运行在本地，也可以是远程服务[reference:5]。
*   **工作原理**：通信前，客户端与服务器会通过“能力协商”明确彼此支持的功能。当AI需要外部信息时，客户端会通过标准化的MCP原语（工具/资源/提示词）请求服务器，服务器处理后返回结果。主机再将结果整合给AI，生成最终答案[reference:6][reference:7]。

### 📞 如何调用MCP？

调用MCP主要通过两种角色实现，你可以根据需求选择：

1.  **作为用户：在应用中配置MCP服务器**
    这是最直接的体验方式，无需编程。你只需在你常用的AI应用（如Claude Desktop、Cursor等）中，通过JSON配置文件添加你需要的MCP服务器即可[reference:8]。配置后，AI就能在对话中自动发现并调用这些工具[reference:9]。
    例如，在Cursor中，你可以在设置里找到MCP配置项，添加类似这样的内容：
    ```json
    {
      "mcpServers": {
        "filesystem": {
          "command": "npx",
          "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/allowed/files"]
        }
      }
    }
    ```

2.  **作为开发者：通过SDK开发客户端和服务器**
    *   **开发客户端**：如果你在构建自己的AI应用，可以使用各语言的MCP SDK来创建客户端，负责连接服务器、发送请求和处理响应。代码会处理复杂的连接和消息路由。
    *   **开发服务器**：使用SDK创建MCP服务器来暴露特定功能。你需要定义你的服务器能提供哪些“工具”(Tools)或“资源”(Resources)，并实现相应的业务逻辑。
        *   下面的Python示例展示了如何用`fastmcp`快速创建一个提供计算工具的服务器[reference:10]：
            ```python
            from mcp.server.fastmcp import FastMCP

            mcp = FastMCP("Demo")

            # 注册一个加法工具
            @mcp.tool()
            def add(a: int, b: int) -> int:
                return a + b

            if __name__ == "__main__":
                mcp.run(transport='stdio')
            ```
