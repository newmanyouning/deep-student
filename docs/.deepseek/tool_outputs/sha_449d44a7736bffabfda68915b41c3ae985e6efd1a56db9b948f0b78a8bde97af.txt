    32│ ### 🚀 一、搭建WPS加载项的基础开发环境
    33│

    35│ 
    36│ *   **安装Node.js**：建议使用 `Node.js v20 LTS` 这类长期支持版本，稳定性更好[reference:0]。
    37│ *   **安装开发工具**：推荐使用代码编辑器 **Visual Studio Code (VS Code)**[reference:1]。
    38│ *   **安装WPS客户端**：确保已安装 **WPS Office 2019 或更高版本**。注意，个人版从特定版本开始对加载项加载有限制[reference:2]，若遇问题可参考此处的[问题排查指南](https://qn.cache.wpscdn.cn/encs/doc/office_v19/topics/WPS%20集成模式/WPS%20加载项开发/加载项概述.html#faq)。

    40│ *   **初始化并调试项目**：安装完成后，通过 `wpsjs create 你的项目名` 创建新项目，根据提示选择加载项类型和Vue/React等前端框架[reference:5][reference:6]。进入项目目录后，运行 `wpsjs debug` 启动调试，该命令会自动打开WPS并加载你的插件[reference:7]。
