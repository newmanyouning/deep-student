    28│ 在 Windows 命令行中，最直接的方法是通过 SSH 连接并使用 `scp` 命令（SSH内置的安全文件复制协议）来下载文件 [citation:1][citation:4][citation:10]。

    30│ 下面是具体操作步骤，分为**前提准备**、**连接并下载**和**常用操作示例**三个部分。
    31│ 
    32│ ### 准备工作：确认基础环境
    33│ 在开始之前，请确保你已经准备好了以下信息，并且本地环境支持 `scp` 命令 [citation:3][citation:6]：
    34│ 1.  **服务器信息**：远程服务器的 IP 地址或域名、SSH 端口号（默认是22）、登录用户名和密码。
    35│ 2.  **本地环境**：Windows 系统（建议 Windows 10 或更高版本，已内置 OpenSSH 客户端）。你可以打开命令提示符 (cmd) 或 PowerShell，输入 `scp` 并回车，如果显示命令用法说明，则说明环境已就绪 [citation:1][citation:10]。
    36│ 
    37│ ### 核心操作：使用 SCP 命令下载文件
    38│ `scp` 命令的下载语法如下：
    39│ ```bash
    40│ scp [可选参数] 用户名@服务器IP:服务器上的文件路径 本地保存路径
    41│ ```
    42│ 
    43│ **关键参数说明 [citation:1][citation:10]**：
    44│ *   `-P 端口号`：如果服务器的 SSH 端口不是默认的22，需要用这个参数指定。
    45│ *   `-r`：如果要下载的是一个**文件夹**（目录），必须加上此参数进行递归复制。
    46│ *   `-C`：启用压缩，传输大文件时能提高速度。
    47│ 
    48│ ### 常见操作示例
    49│ 假设你要连接的服务器信息如下，我会用这个例子来演示几种常见情况：
    50│ *   **服务器IP**：`192.168.1.100`
    51│ *   **用户名**：`root`
    52│ *   **文件路径**：`/home/user/data/report.log`
    53│ *   **下载到本地**：`C:\Users\YourName\Downloads\`
    54│ 
    55│ **1. 下载单个文件到指定目录 [citation:1][citation:7]**
    56│ 打开命令提示符或PowerShell，输入：
    57│ ```bash
    58│ scp root@192.168.1.100:/home/user/data/report.log C:\Users\YourName\Downloads\
    59│ ```
    60│ 输入密码后，文件就会开始传输。
    61│ 
    62│ **2. 下载整个文件夹 [citation:4][citation:8]**
    63│ 如果需要下载 `data` 这个文件夹，一定要加上 `-r` 参数：
    64│ ```bash
    65│ scp -r root@192.168.1.100:/home/user/data C:\Users\YourName\Downloads\
    66│ ```
    67│ 
    68│ **3. 指定非默认端口下载 [citation:10]**
    69│ 如果服务器 SSH 端口改成了 `2222`，需要用大写的 `-P` 指定：
    70│ ```bash
    71│ scp -P 2222 root@192.168.1.100:/home/user/data/report.log C:\Users\YourName\Downloads\
    72│ ```
    73│ 
    74│ ### 备选方案：使用 SFTP 交互式操作
    75│ 如果你希望在连接后，像操作 FTP 一样浏览服务器上的文件再决定下载什么，可以使用 `sftp` 命令 [citation:1][citation:7]。
    76│ 
    77│ 1.  **建立连接**：
    78│     ```bash
    79│     sftp root@192.168.1.100
    80│     ```
    81│     输入密码后，你会进入 `sftp>` 提示符界面。
    82│ 
    83│ 2.  **浏览并下载** [citation:1]：
    84│     *   使用 `ls` 查看服务器上的文件列表，用 `cd` 切换目录。
    85│     *   找到目标文件后，使用 `get` 命令下载：
    86│         ```bash
    87│         # 下载文件到本地当前目录
    88│         get report.log
    89│         
    90│         # 下载整个文件夹
    91│         get -r data
    92│         
    93│         # 指定保存路径
    94│         get report.log C:\Users\YourName\Downloads\
    95│         ```
    96│     *   操作完毕，输入 `bye` 或 `exit` 退出。
    97│
