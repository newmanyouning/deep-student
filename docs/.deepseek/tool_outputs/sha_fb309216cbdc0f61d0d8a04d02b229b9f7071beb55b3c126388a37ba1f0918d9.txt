    34│ ### 🛠️ 方案一：自己动手，将PaddleOCR封装成本地API服务
    35│ 这是最灵活、免费的方式。你可以利用 Python 的轻量级 Web 框架，在几分钟内将 PaddleOCR 打包成一个API服务，供其他程序调用。
    36│ 
    37│ - **核心技术**：使用 **FastAPI** [citation:1] 或 **Flask** [citation:3] 框架搭建 HTTP 服务。
    38│ - **实现思路**：创建一个接口（如 `/ocr`），接收上传的图片，调用 PaddleOCR 识别，最后以 JSON 格式返回结果。
    39│ 
    40│ 下面是一个使用 Flask 搭建的极简示例[citation:3]：
    41│ ```python
    42│ from flask import Flask, request, jsonify
    43│ from paddleocr import PaddleOCR
    44│ import base64
    45│ from PIL import Image
    46│ import io
    47│ 
    48│ app = Flask(__name__)
    49│ # 初始化OCR模型（只需加载一次）
    50│ ocr = PaddleOCR(use_angle_cls=True, lang="ch")
    51│ 
    52│ @app.route('/ocr', methods=['POST'])
    53│ def ocr_api():
    54│     # 1. 获取并解析请求中的图片
    55│     data = request.get_json()
    56│     img_base64 = data['image'].split(',')[-1] # 处理base64格式
    57│     img_data = base64.b64decode(img_base64)
    58│     img = Image.open(io.BytesIO(img_data))
    59│ 
    60│     # 2. 调用PaddleOCR执行识别
    61│     result = ocr.ocr(img, cls=True)
    62│ 
    63│     # 3. 整理并返回结果
    64│     texts = []
    65│     for line in result:
    66│         for word_info in line:
    67│             texts.append(word_info[1][0]) # 提取文本内容
    68│ 
    69│     return jsonify({'texts': texts})
    70│ 
    71│ if __name__ == '__main__':
    72│     app.run(host='0.0.0.0', port=5000)
    73│ ```
    74│ 将这段代码保存为 `app.py` 并运行，你就拥有了一个本地的OCR API服务。你可以通过发送 POST 请求到 `http：//127.0.0.1：5000/ocr` 来识别图片。
