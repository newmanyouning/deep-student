在WPS表格中显示毫秒并进行时间加减操作，需通过以下步骤实现：

---

### **一、显示毫秒且不自动取整**
1. **设置单元格格式**：
   - 选中需要显示毫秒的单元格。
   - 右键选择 **“设置单元格格式”** → **“自定义”**。
   - 在类型中输入格式代码：  
     ```
     hh:mm:ss.000
     ```
   - 点击确定，单元格将显示毫秒（如 `12:34:56.789`）。

2. **输入时间值**：
   - 直接输入时间时，需使用文本格式或公式解析，例如：
     ```
     ="12:34:56.789"
     ```
   - 或使用公式拆分时间（假设时间文本在A1）：
     ```excel
     =TIMEVALUE(LEFT(A1, 8)) + RIGHT(A1,3)/86400000
     ```
     - `86400000` 是一天的毫秒数（24*60*60*1000）。

---

### **二、实现时间加减运算**
1. **加减固定时间段**：
   - 若要加减 `X毫秒`，需转换为小数天数：
     ```excel
     =原时间 + X/86400000
     ```
     例如，加500毫秒：
     ```excel
     =A1 + 500/86400000
     ```

2. **计算两个时间的差值**：
   - 直接相减，设置结果单元格格式为 `hh:mm:ss.000`：
     ```excel
     =结束时间 - 开始时间
     ```

3. **复杂时间加减**：
   - 若需加减 `时:分:秒.毫秒`，使用公式解析：
     ```excel
     =A1 + TIME(小时, 分钟, 秒) + 毫秒/86400000
     ```

---

### **三、示例演示**
1. **输入时间**：
   - A1输入文本 `12:34:56.789`，B1用公式转为数值：
     ```excel
     =TIMEVALUE(LEFT(A1,8)) + RIGHT(A1,3)/86400000
     ```
   - 设置B1格式为 `hh:mm:ss.000`。

2. **加减运算**：
   - 加2秒300毫秒：
     ```excel
     =B1 + 2/86400 + 300/86400000
     ```
   - 或直接：
     ```excel
     =B1 + TIME(0,0,2) + 300/86400000
     ```

---

### **注意事项**
- WPS的时间系统基于数值计算，整数部分为日期，小数部分为时间。
- 若直接输入 `12:34:56.789` 可能被识别为文本，需通过公式转换。
- 毫秒显示依赖自定义格式，确保输入值为正确的时间数值。

通过以上方法，即可在WPS表格中精确处理毫秒级时间并进行运算。
