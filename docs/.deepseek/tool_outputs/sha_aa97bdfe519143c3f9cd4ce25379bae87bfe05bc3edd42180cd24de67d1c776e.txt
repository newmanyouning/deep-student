  1780│ ### 五、完整的一键安装脚本
  1781│ 
  1782│ 创建 `setup-auto-commit.sh`：
  1783│ ```bash
  1784│ #!/bin/bash
  1785│ echo "====== Git自动提交系统安装 ======"
  1786│ 
  1787│ # 1. 创建脚本
  1788│ cat > git-auto-commit.sh << 'EOF'
  1789│ #!/bin/bash
  1790│ while true; do
  1791│     git add . 2>/dev/null
  1792│     if ! git diff --cached --quiet; then
  1793│         git commit -m "自动提交: $(date '+%H:%M:%S')" --quiet
  1794│     fi
  1795│     sleep 30
  1796│ done
  1797│ EOF
  1798│ 
  1799│ # 2. 创建启动脚本
  1800│ cat > start-auto.sh << 'EOF'
  1801│ #!/bin/bash
  1802│ nohup ./git-auto-commit.sh > .auto-commit.log 2>&1 &
  1803│ echo "PID: $!" > .auto-pid
  1804│ echo "✅ 自动提交已启动"
  1805│ echo "📋 命令:"
  1806│ echo "  查看日志: tail -f .auto-commit.log"
  1807│ echo "  停止: pkill -f git-auto-commit"
  1808│ EOF
  1809│ 
  1810│ # 3. 创建停止脚本
  1811│ cat > stop-auto.sh << 'EOF'
  1812│ #!/bin/bash
  1813│ pkill -f git-auto-commit.sh
  1814│ rm -f .auto-pid
  1815│ echo "✅ 自动提交已停止"
  1816│ EOF
  1817│ 
  1818│ # 4. 设置权限
  1819│ chmod +x git-auto-commit.sh start-auto.sh stop-auto.sh
  1820│ 
  1821│ # 5. 更新.gitignore
  1822│ if ! grep -q ".auto-commit.log" .gitignore; then
  1823│     echo ".auto-commit.log" >> .gitignore
  1824│     echo ".auto-pid" >> .gitignore
  1825│ fi
  1826│ 
  1827│ echo "✨ 安装完成！"
  1828│ echo ""
  1829│ echo "👉 使用方法:"
  1830│ echo "  启动: ./start-auto.sh"
  1831│ echo "  停止: ./stop-auto.sh"
  1832│ echo "  查看状态: ps aux | grep git-auto-commit"
  1833│ echo "  查看日志: tail -f .auto-commit.log"
  1834│ ```

  1889│ 
  1890│ 对于初学者，**最简方案**：
  1891│ 1. 创建 `git-auto-commit.sh`（内容用最简单的版本）
