    35│ 
    36│ ### Core
    37│ 
    38│ - **Interactive REPL** with streaming output, command history, slash command autocomplete
    39│ - **Agentic tool loop** — Claude calls tools autonomously until the task is complete
    40│ - **9 built-in tools**: `Read`, `Edit`, `Write`, `Glob`, `Grep`, `Bash`, `AskUser`, `EnterPlanMode`, `ExitPlanMode`
    41│ - **Plan mode** — parallel subagents explore codebase before you implement, with permission isolation
    42│ - **Permission system** — mode-aware (default/plan), reads auto-approved, writes/bash ask for confirmation
    43│ - **Session persistence** — auto-save conversations, `/resume` to continue later
    44│ - **Context compression** — auto-compact when approaching token limits
    45│ - **Anthropic + OpenAI compatible** — works with any compatible API endpoint

   135│ 
   136│ | Tool | Description | Permission |
   137│ |------|-------------|------------|
   138│ | `Read` | Read file contents | auto-approved |
   139│ | `Glob` | Find files by pattern | auto-approved |
   140│ | `Grep` | Search file contents | auto-approved |
   141│ | `Edit` | Edit file (string replacement) | requires confirmation |
   142│ | `Write` | Write/create file | requires confirmation |
   143│ | `Bash` | Run shell command | requires confirmation |
   144│ | `AskUser` | Ask user a question | auto-approved |
   145│ | `EnterPlanMode` | Enter plan mode | auto-approved |
   146│ | `ExitPlanMode` | Exit plan mode | auto-approved |
