### 妙控键盘核心使用技巧摘要：

这里也为你总结一些最常用和最重要的功能：

1.  **连接与充电**：
    *   **连接**：通过磁力吸附到 iPad 侧面的 Smart Connector 上，无需蓝牙配对，即连即用。
    *   **充电**：通过侧面的 USB-C 接口为妙控键盘充电，同时这个接口也能**直通充电**给你的 iPad。

2.  **触控板手势**：
    *   **单指点击**：相当于手指轻点屏幕。
    *   **单指双击并拖动**：移动光标和选择文本。
    *   **双指轻点**：相当于右键点击，调出上下文菜单。
    *   **双指上下滑动**：滚动页面。
    *   **双指张开/捏合**：放大或缩小。
    *   **双指左右滑动**：在 Safari 等应用中前进或后退。
    *   **三指左右滑动**：在最近打开的应用之间快速切换。
    *   **三指上滑**：返回主屏幕或进入 App 切换器。

3.  **键盘快捷键**：
    *   **Cmd (⌘) + H**：返回主屏幕。
    *   **Cmd (⌘) + 空格键**：调出搜索栏。
    *   **Cmd (⌘) + Tab**：在打开的 App 之间切换。
    *   **Cmd (⌘) + C / V / X**：复制/粘贴/剪切。
    *   **长按 Cmd (⌘) 键**：可以查看当前 App 支持的所有快捷键。

4.  **浮动设计**：
    *   妙控键盘采用悬浮式设计，你可以轻松调整角度，找到最适合的观看和打字姿势。

**总结一下，最推荐的方法是访问 Apple 官方的在线使用手册或打开 iPad 自带的“提示”App，它们能为你提供最准确和最新的信息。**
