    44│ ## 方法一：使用 Python 脚本（推荐，跨平台）
    45│ 
    46│ 创建一个 Python 脚本 `merge_md.py`，放在包含这些文件的文件夹中（或指定路径），然后运行。
    47│ 
    48│ ```python
    49│ import os
    50│ import re
    51│ 
    52│ def merge_files(source_dir, output_file):
    53│     # 匹配文件名模式：test_<数字>_preprocessed_img.md
    54│     pattern = re.compile(r'^test_(\d+)_preprocessed_img\.md$')
    55│     files = []
    56│ 
    57│     # 遍历文件夹，找出所有匹配的文件
    58│     for filename in os.listdir(source_dir):
    59│         match = pattern.match(filename)
    60│         if match:
    61│             num = int(match.group(1))
    62│             files.append((num, filename))
    63│ 
    64│     if not files:
    65│         print("未找到匹配的文件。")
    66│         return
    67│ 
    68│     # 按数字升序排序
    69│     files.sort(key=lambda x: x[0])
    70│ 
    71│     # 合并写入输出文件
    72│     with open(output_file, 'w', encoding='utf-8') as outfile:
    73│         for num, filename in files:
    74│             filepath = os.path.join(source_dir, filename)
    75│             with open(filepath, 'r', encoding='utf-8') as infile:
    76│                 content = infile.read()
    77│             # 可选：在每个文件内容前添加一个注释标记（方便阅读）
    78│             outfile.write(f'<!-- 文件: {filename} -->\n\n')
    79│             outfile.write(content)
    80│             outfile.write('\n\n')  # 文件之间空两行
    81│ 
    82│     print(f"成功合并 {len(files)} 个文件到 {output_file}")
    83│ 
    84│ if __name__ == '__main__':
    85│     # 将当前目录作为源目录，输出文件名为 merged.md
    86│     merge_files('.', 'merged.md')
    87│ ```
    88│ 
    89│ **使用方法**：
    90│ 1. 将上述代码保存为 `merge_md.py`。
    91│ 2. 把脚本放在包含那些 md 文件的文件夹中（或修改 `source_dir` 为实际路径）。
    92│ 3. 在终端/命令行中运行：`python merge_md.py`。
    93│ 4. 合并后的文件将生成在相同目录下，名为 `merged.md`。
