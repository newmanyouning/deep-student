    32│ ### 🚀 方法一：使用 `hf download` 命令（最推荐）
    33│ 
    34│ 这是 Hugging Face 从2025年开始主推的新一代命令行工具，命令更规范，支持断点续传，对大模型下载更友好[citation:9]。
    35│ 
    36│ **1. 安装工具**
    37│ 
    38│ 首先，确保你已经安装了 `huggingface_hub` 库，并附带CLI功能[citation:6][citation:9]：
    39│ ```bash
    40│ pip install -U "huggingface_hub[cli]"
    41│ ```
    42│ 
    43│ **2. 运行下载命令**
    44│ 
    45│ 使用 `hf download` 命令，通过 `--local-dir` 参数指定你想保存模型的本地路径[citation:7][citation:9]：
    46│ ```bash
    47│ hf download <模型ID或仓库名> --local-dir <你的本地路径>
    48│ ```
    49│ - `<模型ID或仓库名>`：在模型页面上可以找到，例如 `Qwen/Qwen2.5-7B`。
    50│ - `<你的本地路径>`：你想把模型保存在哪里，例如 `./my-local-model`。
    51│ 
    52│ **举个例子**，将 `Qwen/Qwen2.5-7B` 模型下载到当前目录下的 `qwen-model` 文件夹中：
    53│ ```bash
    54│ hf download Qwen/Qwen2.5-7B --local-dir ./qwen-model
    55│ ```
    56│ 
    57│ > **小贴士**：如果下载中断，再次运行同样的命令即可继续下载[citation:7]。
    58│ 
    59│ ### 📦 方法二：使用 `git clone` 命令（传统方法）
    60│ 
    61│ 这种方法会将整个模型仓库（包括Git历史）克隆下来，适合需要版本控制或进行模型开发的场景[citation:9]。
    62│ 
    63│ 在使用前，请务必确保系统已安装 **Git LFS** (Large File Storage)，因为模型的大文件通过它来管理[citation:1][citation:2][citation:5]：
    64│ ```bash
    65│ # 以Ubuntu/Debian为例安装Git LFS
    66│ sudo apt-get install git-lfs
    67│ git lfs install
    68│ ```
    69│ 
    70│ 安装好Git LFS后，就可以像克隆普通仓库一样克隆模型了[citation:1][citation:2][citation:4]：
    71│ ```bash
    72│ git clone https://huggingface.co/<模型ID或仓库名>
    73│ ```
    74│ 例如，下载 `bert-base-uncased` 模型：
    75│ ```bash
    76│ git clone https://huggingface.co/bert-base-uncased
    77│ ```
