    34│ ### 🎯 针对树莓派5+Ubuntu的选择建议
    35│ 
    36│ 你的设备是**树莓派5 (ARMv8 / AArch64架构)** 运行**Ubuntu系统 (基于Debian)**，因此：
    37│ 1.  **架构选择**：必须选择 **ARM64** (或 AArch64) 版本[citation:5][citation:9]。
    38│ 2.  **格式选择**：Ubuntu系统有**两种简单的方法**，按推荐度排序：
    39│ 
    40│ *   **首选方案：通过系统的APT包管理器安装（最简单）**
    41│     这是最推荐的方法，系统会自动处理安装、更新和依赖关系。
    42│     ```bash
    43│     # 更新软件包列表
    44│     sudo apt update
    45│     # 安装OpenJDK（例如安装JDK 17）
    46│     sudo apt install openjdk-17-jdk
    47│     # 安装后验证版本
    48│     java -version
    49│     ```
    50│     执行`java -version`，如果输出信息中包含 **“aarch64”** 字样，就说明安装的是正确的ARM64版本[citation:9]。
    51│ 
    52│ *   **备用方案：下载ARM64 Compressed Archive（更灵活）**
    53│     如果你需要官网的特定发行版（如Oracle JDK、Eclipse Temurin）或特定版本，可以手动下载这个格式。
    54│     *   **优点**：版本选择完全自主。
    55│     *   **缺点**：需要自己解压、配置环境变量（如`JAVA_HOME`），更新也需要手动操作[citation:5]。

    60│ 2.  **执行安装**：如果通过APT安装，直接运行上面的命令即可。系统仓库中的`openjdk`包已经是针对ARM64优化过的[citation:9]。
    61│ 3.  **验证安装**：安装后务必运行 `java -version`，确保输出信息中同时包含 **“aarch64”** 和预期的版本号。

   104│ ### **第一步：确认操作系统 - 必须是 Linux**
   105│ - ✅ **正确的文件名特征**：名称中一定包含 **“Linux”**。
   106│ - ❌ 错误的名称：包含 “Windows”、 “macOS”、 “AIX” 等。

   108│ ### **第二步：确认CPU架构 - 必须是 ARM64 (AArch64)**
   109│ 这是**最关键的一步**，因为树莓派5的CPU是ARM架构。
   110│ - ✅ **正确的文件名特征**：名称中包含 **`aarch64`** 或 **`ARM64`**。
   111│   - 例如：`...-linux-**aarch64**_bin.tar.gz` 或 `...-**ARM64**_Debian_bin.deb`
   112│ - ❌ 错误的名称：包含 `x64`、 `x86_64`、 `amd64` (适用于Intel/AMD电脑)，或 `arm32`、 `armhf` (适用于旧款树莓派)。

   114│ ### **第三步：确认文件格式 - 优先选择 .tar.gz 或 .deb**
   115│ 根据你系统的包管理器决定：
   116│ 1.  **`.tar.gz` 压缩包 (通用格式)**：
   117│     - **特征**：文件名以 **`.tar.gz`** 或 **`.tgz`** 结尾。
   118│     - **评价**：**最推荐**。它适用于所有Linux发行版，包括Ubuntu。你只需要解压并配置环境变量即可。
   119│ 2.  **`.deb` 包 (Debian/Ubuntu原生格式)**：
   120│     - **特征**：文件名以 **`.deb`** 结尾。
   121│     - **评价**：同样**非常适合**Ubuntu。你可以使用 `sudo dpkg -i 包名.deb` 命令直接安装，系统会自动处理。
   122│ 3.  **`.rpm` 包 (Red Hat系格式)**：
   123│     - **特征**：文件名以 **`.rpm`** 结尾。
   124│     - **评价**：适用于Fedora、CentOS等。虽然Ubuntu可以通过`alien`工具转换安装，但过程繁琐，**不推荐**。
