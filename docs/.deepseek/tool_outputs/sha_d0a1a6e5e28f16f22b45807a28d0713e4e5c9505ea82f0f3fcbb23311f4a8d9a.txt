    26│ 在阿里云服务器上配置好环境后，使用 PaddleOCR 进行识别主要有两种方式：**命令行（快速体验）**和 **Python 脚本（灵活集成）**。你可以根据“简单看看效果”还是“嵌入到自己的程序”来选择。
    27│ 
    28│ ### 💻 方式一：命令行快速识别（推荐新手）
    29│ 
    30│ 如果你只是想快速测试一下效果，这是最直接的方法。打开服务器的终端，使用 `paddleocr` 命令即可。
    31│ 
    32│ 1.  **识别单张图片**：
    33│     运行以下命令，工具会自动下载默认模型并输出识别结果。
    34│     ```bash
    35│     paddleocr --image_dir /path/to/your/document.jpg --use_angle_cls true
    36│     ```
    37│     *   `--image_dir`：替换为你上传的文档图片路径。
    38│     *   `--use_angle_cls true`：启用方向分类，能自动校正旋转的图片，建议加上 [citation:9]。
    39│ 
    40│ 2.  **识别整个文件夹**：
    41│     如果你想批量处理，直接指定文件夹路径，它会处理下面所有支持的图片。
    42│     ```bash
    43│     paddleocr --image_dir /path/to/your/images/folder --use_angle_cls true
    44│     ```
    45│ 
    46│ 3.  **识别 PDF 文件**：
    47│     如果你的文档是 PDF，也直接支持，还可以用 `--page_num` 指定只识别前几页。
    48│     ```bash
    49│     paddleocr --image_dir /path/to/your/doc.pdf --use_angle_cls true --page_num 3
    50│     ```
    51│     *   `--page_num`：默认为0，表示识别所有页 [citation:9]。
    52│ 
    53│ ### 🐍 方式二：Python 脚本调用（推荐开发集成）
    54│ 
    55│ 如果你想在代码中进一步处理识别结果（比如保存到数据库、分析文本），或者在Web应用里调用，用Python脚本会更灵活。
    56│ 
    57│ 1.  **创建一个 Python 文件**（例如 `my_ocr.py`），写入以下代码：
    58│     ```python
    59│     from paddleocr import PaddleOCR
    60│ 
    61│     # 1. 初始化 OCR 引擎（只需执行一次）
    62│     # 假设你需要识别中文文档
    63│     ocr = PaddleOCR(use_angle_cls=True, lang='ch')
    64│     # 参数说明：
    65│     # use_angle_cls=True: 启用方向分类
    66│     # lang='ch': 使用中文模型。可选 'en', 'japan', 'korean' 等 [citation:3]
    67│ 
    68│     # 2. 指定你的文档图片路径
    69│     img_path = '/path/to/your/document.jpg'
    70│ 
    71│     # 3. 执行识别
    72│     # cls=True 表示在识别过程中也使用方向分类
    73│     result = ocr.ocr(img_path, cls=True)
    74│ 
    75│     # 4. 解析和打印结果
    76│     # 结果是一个列表，每个元素对应一个检测到的文本框
    77│     for line in result:
    78│         # line[0] 是文本框的四个角坐标
    79│         # line[1][0] 是识别的文本内容
    80│         # line[1][1] 是识别的置信度
    81│         print(f"文本: {line[1][0]}, 置信度: {line[1][1]:.2f}")
    82│     ```
    83│ 
    84│ 2.  **运行脚本**：
    85│     ```bash
    86│     python my_ocr.py
    87│     ```
