   207│ ### 📏 各版本上下文规格
   208│ *   **Claude Opus 4.6 / Sonnet 4.5 / Sonnet 4**：支持 **100万 tokens**（约75万英文单词，可处理超7.5万行代码），需特定方式启用[citation:2][citation:4]。
   209│ *   **标准 Claude.ai 付费版**：**20万 tokens**（约15万英文单词），这是大多数用户日常使用的默认上限[citation:6][citation:10]。
   210│ *   **早期版本（如 3.7 Sonnet）**：**20万 tokens**，目前已逐步被新版覆盖[citation:1]。

   212│ ### 🚀 如何启用 100万 超长上下文？
   213│ 不同平台的启用方式略有不同：

   216│ *   **Cloud / Console**：在模型选择下拉菜单中，直接选择带有 **“1M (beta)”** 标识的模型变体即可[citation:4]。
   217│ *   **Claude Code**：已默认集成，无需额外配置[citation:2]。
   218│
