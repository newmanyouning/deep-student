### 知识点1: VBA代码在Excel文件中的存储位置
VBA代码以二进制数据形式嵌入在启用了宏的Excel文件（如 `.xlsm`、`.xlsb`）中。从文件结构看，`.xlsm` 文件本质是一个 ZIP 压缩包，VBA 项目部分打包在一个名为 **`vbaProject.bin`** 的二进制文件里。在逻辑上，代码被组织在 VBA 项目中，包含以下部分：标准模块（存放通用宏代码）、类模块（存放类定义）、窗体（存放自定义用户界面）、工作表对象（存放与特定工作表相关的代码）、ThisWorkbook 对象（存放与整个工作簿相关的代码）。

### 知识点2: 使用内置VBA编辑器查看和编辑代码
最直接的方法是通过 Excel 自带的 VBA 编辑器。操作步骤：1. 打开包含宏的 Excel 文件；2. 按下快捷键 **`Alt + F11`** 打开 VBA 编辑器；3. 在左侧的"工程资源管理器"（Project Explorer）中找到 VBA 项目，双击模块即可查看和编辑代码。

### 知识点3: 使用外部编辑器编辑VBA代码
可将 VBA 代码导出为纯文本文件后用外部编辑器（如 VS Code、Sublime Text）编辑。**导出**：在 VBA 编辑器中右键点击要导出的模块，选择"导出文件"，保存为 `.bas`（标准模块）、`.cls`（类模块）或 `.frm`（窗体）文件。**导入**：在 VBA 编辑器中右键点击"模块"文件夹，选择"导入文件"，选择修改后的文件即可。

### 知识点5: 通过Python GUI自动化操作WPS VBA编辑器
使用 `pywinauto` 库模拟鼠标和键盘操作来自动化 WPS 的 VBA 编辑流程。核心步骤：启动 WPS、按下 `Alt+F11` 打开 VBA 编辑器、插入模块、粘贴代码、保存。示例代码：`app = Application(backend="uia").start(r"C:\path\to\wps.exe")`，然后通过 `app.window(title_re=".*WPS.*").type_keys("%{F11}")` 打开编辑器。**缺点**：脚本较脆弱，WPS 版本更新或窗口变化可能导致脚本失效，且需额外处理"启用宏"等安全弹窗。
