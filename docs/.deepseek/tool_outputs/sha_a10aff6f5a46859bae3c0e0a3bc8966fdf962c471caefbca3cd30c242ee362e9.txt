   253│ ### 二、核心方法：**Potential-Dependent Surrogate Gradients (PDSG)**  
   254│ #### ❓ **为什么需要PDSG？**  
   255│ SNN的脉冲发放函数（如 **LIF神经元**）是阶跃函数，其导数几乎处处为零（如图1左），导致反向传播时梯度消失：  
   256│ > `梯度 = ∂损失/∂脉冲 ≈ 0` → 无法生成对抗扰动  
   257│ 
   258│ 传统代理梯度（如矩形函数、Sigmoid导数）是**固定函数**（图1中），但忽略了神经元**膜电位动态**（随时间变化的电压值），导致梯度与真实状态不匹配。  
   259│ 
   260│ #### 💡 **PDSG的解决方案**  
   261│ PDSG的核心思想：**用膜电位（Neuron Membrane Potential）动态调节代理梯度**：  
   262│ > **代理梯度函数** = `f(V) = λ * sech²(β(V - V_th))`  
   263│ > **其中**：  
   264│ > - `V`：当前时间步的膜电位（随时间累积输入）  
   265│ > - `V_th`：发放阈值  
   266│ > - `λ`, `β`：可学习参数，控制梯度幅值与平滑度  
   267│ 
   268│ **1. 膜电位依赖的梯度计算（图1右）**  
   269│    - **当 `V` 接近阈值 `V_th` 时**：梯度值大（`sech²(0)=1`），反映脉冲发放临界点的敏感性。  
   270│    - **当 `V` 远离阈值时**：梯度值小，避免噪声干扰。  
   271│    > ✅ **物理意义**：贴合神经元真实发放行为（电位近阈值时微小扰动即可触发脉冲翻转）。  
   272│ 
   273│ **2. 时空梯度聚合**  
   274│    在SNN的时间步 `t` 和层 `l` 上聚合梯度：  
   275│    > `总梯度 = Σ_t Σ_l [ ∂损失/∂脉冲_{t,l} × PDSG(V_{t,l}) ]`  
   276│    - **解决时间维度碎片化**：梯度跨时间步传播，捕捉脉冲时序依赖性。  
   277│ 
   278│ **3. 梯度到脉冲的离散化（G2S）**  
   279│    将连续梯度转为三元脉冲扰动：  
   280│    ```
   281│    if 梯度 > 0.5  → 输出 +1（添加脉冲）
   282│    if 梯度 < -0.5 → 输出 -1（删除脉冲）
   283│    否则 → 0（不修改）
   284│    ```

   293│ | **脉冲翻转效率**   | 需更多扰动           | **近阈值时微小扰动即翻转** |
