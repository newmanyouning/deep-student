1. **Hook 复用度**: 哪些 Hook 被大量使用
2. **Hook 依赖链**: 是否有过深的 Hook 调用链
3. **引擎架构**: 渲染引擎的插件化程度

5. **样式现状**: CSS 文件数量和组织方式，验证文档中 "60+ scattered CSS files" 的描述
