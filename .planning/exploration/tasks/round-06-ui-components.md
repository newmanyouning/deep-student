## 诊断要点

1. **组件变体管理**: 是否使用 CVA / class-variance-authority
2. **Radix UI 封装**: 哪些组件是对 Radix 的封装
3. **无障碍**: 是否有 ARIA 属性、键盘导航
4. **移动端适配**: 是否处理了移动端触摸交互
5. **重复组件**: 是否有功能相似但不同名的组件
